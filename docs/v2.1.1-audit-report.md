# v2.1.1 — Security Audit & Hardening Report

## Block A — Security

### A1. Dependency Audit
- **Production deps:** `npm audit --omit=dev` → 1 high (d3-color <3.1.0, ReDoS), transitive via recharts. No direct fix (would require recharts major bump). Low impact — only triggers on crafted color strings in charts. **Accepted.**
- **License check:** All deps MIT/Apache-2.0/BSD/ISC. Zero GPL/AGPL/LGPL. ✅

### A2. Defang & Data Leakage
- **Scan:** All `data/normalized/*.json` files scanned for real domains, real IPs, brand mentions, PII, API keys.
- **Result:** Zero violations. All domains use `.example.tk/.ml/.cf/.ga/.tr`. All IPs use RFC 5737 TEST-NET ranges. ✅

### A3. XSS Surface
- **`dangerouslySetInnerHTML`:** 1 usage in `Detections.tsx` (YAML highlighter). Input is sanitized via `escape()` in `highlightYAML()`. Safe. ✅
- **`innerHTML`:** Zero. ✅
- **`eval` / `new Function`:** Zero. ✅

### A4. Content Security Policy
- **Meta CSP added:** `default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' data: blob:; font-src 'self' https://fonts.gstatic.com data:; connect-src 'self' https://fonts.googleapis.com; frame-ancestors 'none'`
- **`'unsafe-inline'`** needed for Tailwind + SVG lib (Recharts, reactflow, react-simple-maps use inline styles).
- **`'unsafe-eval'` NOT included** — prod build doesn't need it. ✅

### A5. Subresource Integrity
- No CDN scripts/styles. All assets via npm. SRI not applicable. ✅

### A6. Secret Scanning
- Zero secrets found in source code or data files. ✅

### A7. .gitignore
- Covers: node_modules/, dist/, .env*, *.local, .DS_Store. `dist/` not committed (build output). ✅
